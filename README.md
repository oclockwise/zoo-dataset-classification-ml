# 🐾 Zoo Dataset ML Classification

## 📌 Overview
This project applies machine learning models to classify animals based on their features using the Zoo dataset.

## ⚙️ Models Used
- k-Nearest Neighbors (k-NN)
- Naive Bayes
- Decision Tree

## 🔧 Key Steps
- Data preprocessing & scaling  
- Train-test split (70/30)  
- Stratified cross-validation  
- Model comparison  

## 📈 Results
- k-NN and Decision Tree achieved ~90% accuracy  
- Naive Bayes showed consistent performance  

## 🛠️ Tech Stack
Python, pandas, numpy, scikit-learn, matplotlib
